### Introdução

As medias enviadas ou recebidas nas mensagens do whatsapp, são processadas pela a meta. Podendo ser enviadas, recuperadas e visualizadas.

## Manipulação de medias

### Buscar uma media pelo id

Endpoint utilizado para buscar uma media pelo id da media na META:

#### (POST) https://api.positus.global/v2/whatsapp/numbers/{NUMBER_ID}/media-by-id/{ID_MEDIA}

`Response`
